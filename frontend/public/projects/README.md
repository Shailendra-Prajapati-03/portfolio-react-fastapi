# Project images

Local screenshots for the Projects section. Files here are served from the site
root (e.g. `public/projects/ai-lead-validation.png` → `/projects/ai-lead-validation.png`).

## ai-lead-validation.png (n8n workflow)

Save your **n8n dark-mode** workflow screenshot here as `ai-lead-validation.png`.

- Switch n8n to the **dark theme** first (Settings → Appearance / theme) so the
  canvas background is black — this matches the card's black frame with no editing.
- Crop it to roughly a **16:10** aspect ratio (wide) so the card's `object-cover`
  doesn't cut off important nodes.
- PNG or JPG both fine; keep it reasonably sized (≈ 1200px wide).

Once the file is in place it appears automatically — no code change needed.

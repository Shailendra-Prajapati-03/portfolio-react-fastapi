# Public assets

Drop these files here to complete the portfolio. The app degrades gracefully if
they're missing (the hero shows a gradient "SP" monogram instead of a photo).

| File           | Used by            | Purpose                                  |
| -------------- | ------------------ | ---------------------------------------- |
| `avatar.jpg`   | Hero portrait      | Square headshot (≥ 600×600)              |
| `resume.pdf`   | Hero "Download CV" | Your résumé                              |
| `og-image.png` | Social previews    | 1200×630 Open Graph / Twitter card image |
| `favicon.svg`  | Browser tab        | Already included (gradient "S")          |
